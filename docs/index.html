Heytaco Anywhere

Heytaco Anywhere is 3rd party plugin of Heytaco. It connects Heytaco with other services such as Github.

Get started

Let’s install Heytaco Anywhere. In this example, we’re going to connect Heytaco(Slack) with Github. Step 1: Preparation To make Heytaco Anywhere acts on behalf of a Slack user, we have to use Slack OAuth.

Configuration

Heytaco Anywhere is connected by OAuth. It is necessary to configure Slack or Microsoft Teams. It depends on your Heytaco platform. And you should select one of the services to communicate with Heytaco and configure the OAuth setting for App.
