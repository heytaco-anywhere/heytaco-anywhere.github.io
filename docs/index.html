Overview

Heytaco Anywhere

Heytaco Anywhere is 3rd party plugin of Heytaco. It connects Heytaco with other services such as Github.

Get started

Let’s install Heytaco Anywhere. In this example, we’re going to connect Heytaco(Slack) with Github.

Configuration

Heytaco Anywhere is connected by OAuth. It is necessary to configure Slack or Microsoft Teams.

Installation

Basically, Heytaco Anywhere is self-managed service. So it is necesary to install in your infrastructure.
